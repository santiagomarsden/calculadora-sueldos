Importar / exportar configuración

Esto sirve para copiar una configuración entre navegadores o clientes.

Cómo usar las fórmulas: podés agregar o borrar conceptos desde este panel y editar sus fórmulas sin tocar el diseño. Ejemplos: presentismAmount, snrBase + snrSeniority + presentismAmount, grossRem * jubilacion, affiliate como condición, o years > 0 para mostrar algo sólo si corresponde.
